Recognition and Awards

GreatUniHack 2016 Won first prize with our hack Wimphony. Wimphony took Wi-fi SSIDs and RSSIs at various locations and mapped them in a virtual world. It would then generate music based off of the SSID and would place this sound in the virtual world. The user could then explore this world using Google Cardboard. This project taught me a lot about Android development and introduced me to Unity.

Young Rewired State Festival of Code 2014 Won Best In Show (public vote) with a smart coat hanger called Hook. Hook would query the weather each day and light up the hanger holding the appropriate item of clothing for that day. Wrote the back-end that communicated with the Met Office API and configured/managed the Raspberry Pi that powered it.
